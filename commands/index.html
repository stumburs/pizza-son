    pizza_son Commands

    This page lists all* available bot commands.

    !mark

    Description:

    Generates a random message using a basic Markov Chain algorithm. Knowledge is generated from every message sent in this chat.

    Usage:

    !mark

    Permissions:

    All

    !ada

    Description:

    Responds based off previous interactions. Functions similarly to the chatbot - Cleverbot.

    Usage:

    !ada <text>

    Permissions:

    All

    !lobotomize

    Description:

    Clears the AI's memory.

    Usage:

    !lobotomize

    Permissions:

    Streamer, Bot Moderator

    !iq

    Description:

    Shows the IQ of target user.

    Usage:

    !iq [user]

    Permissions:

    All

    !lark

    Description:

    Generates a random message trained on this chat (!mark) and reinterprets the output using an LLM.

    Usage:

    !lark

    Permissions:

    All

    !commands

    Description:

    Sends a link to this page.

    Usage:

    !commands

    Permissions:

    All

    !pee

    Description:

    Pees on the target person.

    Usage:

    !pee <user>

    Permissions:

    All

    !togglebestie

    Description:

    Toggles the 'Bestie' TTS for LLM responses.

    Usage:

    !togglebestie

    Permissions:

    Streamer, Bot Moderator

    !setvoice

    Description:

    Sets the 'Bestie' TTS voice.

    Usage:

    !setvoice <voice>

    Permissions:

    Streamer, Bot Moderator

    !ping

    Description:

    Replies with 'pong!' to test the bot's responsiveness.

    Usage:

    !ping

    Permissions:

    All

    !jark

    Description:

    Generates a random message trained on this chat (!mark) and reinterprets the output using an LLM (using Jon's prompt).

    Usage:

    !jark

    Permissions:

    All

    !game

    Description:

    Replies with the current game being played on this stream.

    Usage:

    !game

    Permissions:

    All

    !hug

    Description:

    Hugs the target user.

    Usage:

    !hug <user>

    Permissions:

    All

    !math

    Description:

    Replies as 'math' using an AI.

    Usage:

    Usage: !math <question>

    Permissions:

    All

    !cat

    Description:

    Replies as 'cat' using an AI.

    Usage:

    Usage: !cat <question>

    Permissions:

    All

    !nsfw

    Description:

    Replies as 'nsfw' using an AI.

    Usage:

    Usage: !nsfw <question>

    Permissions:

    All

    !llm

    Description:

    Replies as 'llm' using an AI.

    Usage:

    Usage: !llm <question>

    Permissions:

    All
